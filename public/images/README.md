# Imágenes de proyectos

Pon aquí las fotos de tus proyectos con exactamente estos nombres:

- misintaxis.png       → screenshot/logo de MiSintaxis app
- misintaxis-web.png   → screenshot de la plataforma web de profesores
- timeslearning.png    → screenshot/logo de TimesLearning
- gymex.png            → screenshot/logo de GymEx

Tamaño recomendado: 200x200px o superior (se recorta en cuadrado).
Formatos soportados: .png, .jpg, .webp

Si no tienes imagen para un proyecto, el portfolio mostrará
automáticamente las iniciales del proyecto (ej: "MS" para MiSintaxis).
